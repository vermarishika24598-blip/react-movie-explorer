# React Movie Explorer 🎬

A responsive and modern **Movie Explorer Web App** built with **React.js**, **Tailwind CSS**, and **Framer Motion**.
Users can explore trending movies, search films by title, and view detailed movie information — all powered by the TMDB API.

## 🚀 Features

* 🔍 **Search movies** by name
* 🎞️ **Browse trending movies**
* 📄 **View movie details** (rating, overview, poster, release date)
* 📱 **Fully responsive UI**
* ✨ **Smooth animations** using Framer Motion

## 🛠 Tech Stack

* React.js
* Tailwind CSS
* Framer Motion
* TMDB API
* React Router

## 🔑 TMDB API Setup

This project uses the TMDB API. To run it locally, create a `.env` file and add your **private API key**.

> The API key is **not included** in this repository for security reasons.

---

## 🖥️ Installation & Setup

**Clone the repository:**

```
git clone https://github.com/vermarishika24598/react-movie-explorer.git
cd react-movie-explorer
```

**Install dependencies:**

```
npm install
```

**Run the development server:**

```
npm run dev
```
