# react-movie-explorer
🎬 React Movie Explorer

A responsive and modern Movie Explorer App built with React.js, Tailwind CSS, and Framer Motion.
Users can explore trending movies, search for films, apply filters, and see detailed movie information using the TMDB API.

🚀 Features

🔍 Search movies by name

🎞️ Browse trending movies

🎭 Filter movies by categories

📄 View movie details (rating, overview, poster, release date)

📱 Fully responsive

✨ Smooth animations using Framer Motion

🛠 Tech Stack

React.js

Tailwind CSS

Framer Motion

TMDB API

React Router

📁 Folder Structure
src/
  components/
  pages/
  hooks/
  App.jsx
  index.js

🔑 TMDB API Setup

Create a .env file in the project:

VITE_TMDB_API_KEY=your_api_key_here

🖥️ Installation & Setup

Clone the repo:

git clone https://github.com/vermarishika24598/react-movie-explorer.git
cd react-movie-explorer


Install dependencies:

npm install


Run the development server:

npm run dev
