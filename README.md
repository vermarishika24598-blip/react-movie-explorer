# React Movie Explorer

A fullstack React application to explore movies, featuring user authentication, watchlists, and browsing popular, upcoming, and top-rated movies.

## Features

- User signup and login
- Create and manage a watchlist
- Browse popular, top-rated, and upcoming movies
- Redux for state management
- Fullstack integration with backend API

## Tech Stack

- **Frontend:** React, Tailwind CSS, Redux
- **Backend:** Node.js, Express
- **Database:** (Add your DB here, e.g., MongoDB)
- **Authentication:** JWT
- **API Integration:** Movie data APIs

## Installation

### 1. Clone the repository
```bash
git clone https://github.com/vermarishika24598-blip/react-movie-explorer.git
cd react-movie-explorer
Backend Setup
cd backend
npm install


Create a .env file in the backend folder and add:

PORT=5000
DB_URI=your_database_uri
JWT_SECRET=your_secret_key


Start the backend server:

npm run dev

Frontend Setup
cd ../frontend
npm install
npm start


Frontend will run at: http://localhost:1234

Usage

Open the frontend at http://localhost:1234

Signup or login to start using the app

Browse movies and add them to your watchlist or favorites

Backend API handles all data and authentication
